JSON CorrectorAUTO-FIX

Keyboard shortcut: Ctrl/Cmd + Enter

Validation Warnings:

    Click "Correct JSON" to see the result...

    Supported Corrections:

    • NULL, Null, None, none → null
    • TRUE, True → true, FALSE, False → false
    • Single quote strings → double quote strings
    • Trailing commas in arrays and objects
    • Unquoted object keys
    • NEW: Ignores line comments: // and #
    • NEW: Auto-fixes unterminated strings
    • NEW: Auto-closes unmatched braces {}
    • NEW: Auto-closes unmatched brackets []
    • NEW: Fixes mismatched brackets
