## 1. 温度为什么"一直不动"?——这是能量守恒,不是 bug

温度的定义(代码里):
```
T = 平均动能 = (1/N) Σ ½ m v²      (取 k=1, m=1)
```

孤立盒子里发生的每件事都**不增减总动能**:直线飞行不变;弹性撞墙只反号一个分量(|v| 不变);弹性碰撞交换速度但总动能守恒。→ 总动能精确守恒 → **温度恒定**。

💡 这正是"孤立理想气体平衡态温度不变"。要让温度动,必须真的注入/取走能量:
- 加热/降温按钮(缩放速度);
- **移动活塞压缩**:撞在运动的墙上 `v' = 2·v墙 − v`,墙对气体做功 → 绝热升温(只在活塞移动过程中变);
- 恒温壁(按墙温重发速度)。

## 2. 压强、体积到底怎么"测"出来

- **体积**其实没测:就是盒子尺寸 `V = 宽 × 高`(2D 是面积)。纯几何。
- **压强**是真测的,按定义"力/面积"来:

```
第一步:每次撞墙,墙受到的冲量 Δp = |v后 − v前|·m      (牛顿第三定律)
         累加到 wallImpulse
第二步:攒一个时间窗后
         P = wallImpulse / (Δt × 周长)
             ÷Δt   → 冲量变成平均力
             ÷周长 → 力变成"单位墙长上的力" = 压强
```

就是模拟一个真实压力计:统计单位时间、单位墙长上分子砸墙的总"推力"。把 P、V、T 一乘除,`PV/(NkT)` 一直贴着 1——**没告诉它气体定律,碰撞规则自己满足**。

## 3. 温度 ↔ 焦耳的换算 = 能量均分定理

$$\langle E_{kin}\rangle = \frac{f}{2}k_B T \qquad(f=\text{平动自由度})$$

- 每个"平方型自由度"分到 **½ k_B T**。
- 3D:⟨E⟩ = (3/2)k_B T;**2D(我那个 demo):⟨E⟩ = k_B T**;1D:½ k_B T。
- **k_B = 1.380649×10⁻²³ J/K**。

反解温度:`T = ⟨E⟩ / (f/2 · k_B)`。

💡 温度本质上就是能量,k_B 只是"开尔文↔焦耳"的汇率。物理里常令 k_B=1,直接拿能量当温度。
💡 记住数字:室温 300 K,每分子 ⟨E⟩ ≈ (3/2)k_B·300 ≈ 6.2×10⁻²¹ J ≈ **25.8 meV**(著名的"室温能标")。v_rms = √(3k_BT/m),氮气 300K ≈ 517 m/s。

---

## 4. k_B 是"每个粒子"的;而且对所有气体都一样

- **k_B = R / N_A**:R(8.314 J/mol·K)是每摩尔的,k_B 是**每个粒子**的。
- 同一定律两种写法:人均 `⟨E⟩=(3/2)k_BT`,每摩尔 `(3/2)RT`,差一个 N_A。
- **k_B 是普适常数**,跟气体种类无关(R 也是,所以叫"通用"气体常数)。

⚠️ 不同气体差的是**质量 m**,不是 k_B。同一温度下所有气体平均动能相同,但**速度不同**:`v_rms = √(f k_B T / m)`,轻的快(氦)、重的慢(氙)。

---

## 5. 同 T、同 P → 同 V(阿伏伽德罗),前提是"同分子数"

$$PV = N k_B T \Rightarrow V = \frac{N k_B T}{P}$$

定律里**没有质量项** → 固定 T、P,体积只由**粒子数 N** 决定,与气体种类无关。
标准状况下 1 摩尔任何理想气体都占 **22.4 L**。

⚠️ **陷阱**:"同体积"对应"同分子数",**不是同质量**。1 克 H₂ 比 1 克 O₂ 体积大得多。

💡 引申:同 T、同 P → 数密度 n=P/(k_BT) 相同 → **平均分子间距也相同**,与气体种类无关。因为理想气体粒子是"无个性的计数单位",只有"存在"被计数(这就是"数原子"、阿伏伽德罗数有意义的原因)。只有分子当成点、彼此不相互作用时成立;使劲压缩到分子快挨上,就失效。

## 6. 压强明明含动量 mv,质量怎么会消失?——被温度吸收了

压强推导(纯牛顿):
```
P = n·m·⟨vx²⟩          (每次撞墙动量∝mv,撞墙频率∝v,密度n)
```
质量确实在里面。但**温度把 m⟨vx²⟩ 钉死了**:

```
能量均分:½ m⟨vx²⟩ = ½ k_B T   ⟹   m⟨vx²⟩ = k_B T
代入:P = n·(m⟨vx²⟩) = n·k_B T          ← 质量消失
```

💡 质量不是"不在",是**被温度收走了**。物理直觉:重分子撞得重($mv$ 大)但撞得慢、撞得少($v$ 小),`m` 变大多少 `⟨v²⟩` 就变小多少,乘积 `m⟨v²⟩=k_BT` 恒定,正好抵消。
- 固定**速度**改质量 → 压强变;
- 固定**温度**改质量 → 压强不变。因为温度测的是能量 $mv²$,不是速度 $v$、也不是动量 $mv$。

## 7. "速度服从玻尔兹曼分布"——能证明,不是猜

平衡分布:`f(v) ∝ exp(−½mv² / k_BT) = exp(−E/k_BT)`(玻尔兹曼因子)。

**最力学的证法(细致平衡 + 碰撞不变量):**
平衡态下正逆碰撞速率平衡,加"分子混沌"(碰前速度不相关):
```
f(v1)·f(v2) = f(v1')·f(v2')
取对数 ⟹ ln f 是"碰撞中守恒的可加量"
```
而弹性碰撞守恒的可加量**只有三个**:粒子数、动量 mv、动能 ½mv²。所以
```
ln f = a + b·(mv) + c·(½mv²)
静止气体 b=0,可归一 c=−1/k_BT
⟹ f(v) ∝ exp(−½mv²/k_BT)          ← 麦克斯韦-玻尔兹曼,被"逼"出来的
```

**会不会收敛到它?** 玻尔兹曼 H 定理:H=∫f ln f dv 单调下降(熵单调上升),只在麦克斯韦分布时停。→ 任意初始分布都被碰撞推向它。**idealgas.html 里点"重置"看直方图从尖峰长成红线,就是 H 定理现场演示。**

其它殊途同归的证法:麦克斯韦对称性论证、正则系综 e^(−E/kT)、最大熵(Jaynes)。四条路同一个答案。

⚠️ 诚实边界:都依赖一条**公设**(等概率原理 / 分子混沌),它是不可逆性的来源(洛施密特佯谬),极合理但非纯逻辑定理。

## 8. 玻尔兹曼 vs 麦克斯韦:一条定律的两个面孔

根本只有一个:**玻尔兹曼因子 e^(−E/k_BT)**。麦克斯韦是它套到"速率"上、再乘几何态密度:

| 变量 | 分布 | 来源 |
|---|---|---|
| 速度**分量** vx | 高斯(对称钟形,中心0) | 纯玻尔兹曼因子 |
| **速率** \|v\| | 麦克斯韦(偏斜,∝v²·e^…) | 玻尔兹曼 × 球壳态密度 4πv² |
| **能量** E | ∝√E·e^(−E/kT) | 玻尔兹曼 × 能量态密度 |

⚠️ 易混:**"钟形"是速度分量,不是速率**。速率是一边倒的偏斜分布(那个 v²/v 因子纯粹是"速率为 v 的方向有多少个"的几何)。

## 9. 碰撞怎么模拟 & 同速率为何分化出分布

**碰撞模拟:**
- 撞墙:垂直分量反号。
- 粒子间:用网格检测重叠;等质量弹性碰撞 = **沿两球连线方向交换速度分量,切向不变**(精确守恒总动量、总动能);再把重叠推开防粘。

**同速率为何会分化(核心)**:碰撞守恒的是**总动能,不是每个粒子的速率**。碰撞只交换连线方向分量,撞击角度随机 → 能量在这一对之间被**不均匀重分配**,一快一慢。全场不停随机重分配,尖峰速率迅速摊开 → 麦克斯韦分布。

💡 类比:一屋子人各 100 元,两两随机交易(守恒总额、重分配)→ 很快出现"贫富分布"。这不是比喻——**加法式随机交换严格给出指数分布(玻尔兹曼-吉布斯)**,和气体是同一套细致平衡数学。钱 ≈ 能量(守恒、可加、非负),⟨钱⟩ 扮演温度。
⚠️ 但真实财富是**幂律长尾(帕累托)**,因为真实交易含乘法/投资效应,不是简单加法交换。

## 10. "平衡态"到底怎么衡量

一句话:**微观一直在变,宏观(分布)不再变**。平衡是分布/宏观态的性质,不是个体的。

怎么判定"到了":
1. 宏观量停止漂移(T、P、PV/NkT、直方图形状只在稳定值附近抖);
2. 分布贴上理论曲线并稳住;
3. **最严格:熵 / H 函数到达平台**(dH/dt=0,熵不再增)。画熵随时间的曲线,压平那刻就是平衡;
4. 细致平衡(正逆碰撞速率相等)。

⚠️ 平衡不是死寂,仍有涨落,幅度 ∝ 1/√N(粒子越多越平稳)。判据是"只有随机抖动、无系统性趋势"。
💡 为什么停在那儿:平衡分布是**熵最大**的宏观态,对应的微观态数目压倒性地多,系统随机游走几乎总待在里面。

## 11. 还有一大票现象服从玻尔兹曼 e^(−E/kT)

把 E 换成对应能量即可:
- **大气密度随高度**(气压公式)E=mgh → n(h)∝e^(−mgh/kT)(佩兰用它测阿伏伽德罗数);
- **反应速率**(阿伦尼乌斯)∝e^(−Ea/kT);
- **原子能级布居**(光谱、激光)激发/基态比 = e^(−ΔE/kT);
- **半导体载流子**、**顺磁性**(居里)、**电池/神经元膜电位**(能斯特)、**沉降**…

**量子表亲**(低温/高密度时):
- 费米-狄拉克(费米子)`1/(e^((E−μ)/kT)+1)`;
- 玻色-爱因斯坦(玻色子)`1/(e^(E/kT)−1)` —— **黑体辐射**(blackbody.html)。
- 稀薄/高温时 ±1 可忽略 → 退回玻尔兹曼。**玻尔兹曼是量子分布的经典极限。**

## 12. 统计力学 vs 热力学(一张表收束)

| | 热力学 | 统计力学 |
|---|---|---|
| 层次 | 宏观、唯象 | 微观→宏观、从力学+概率推导 |
| 回答 | 是什么、什么可能(规则) | 为什么、怎么来的(机制) |
| 温度 | 基本量(测量) | = (2/f)⟨E_kin⟩ 平均动能 |
| 熵 | dS=dQ/T(克劳修斯) | **S = k_B ln W**(玻尔兹曼,数微观态) |
| 历史 | 先(蒸汽机,1820–60) | 后(1860–1900,解释热力学) |

**熵是两者交汇处**:热力学从热量定义它,统计力学揭示它是"微观态数目的对数"。
