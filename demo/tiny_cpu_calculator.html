TinyCPU 计算器

一个从零搭的 128 位定点处理器模拟器（Q64.64，10+ 位有效数字）。屏幕上每一个数字——哪怕是 3+5——都由这颗模拟 CPU 的 ALU 逐位算出，三角函数用 CORDIC 机器码程序（只靠加法和移位）算出。

数字格式：有符号 128 位定点 Q64.64（整数 64 位 + 小数 64 位），分辨率 2⁻⁶⁴≈5.4e-20，足够 10+ 位有效数字。
加/减：128 位纹波进位加法器，逐位逻辑门；减法用二进制补码。
乘/除：移位-相加 / 移位-相减（内部用 256 位累加器，仍归结为上面的门加法器）。
sin/cos：CORDIC 机器码子程序，48 次迭代——只用加法、减法、算术右移，外加一张 arctan(2⁻ⁱ) 只读表。π 等常数以 50 位十进制精度写入 ROM。
√：牛顿迭代 x←(x+a/x)/2（机器码子程序）。tan = sin ÷ cos。角度制先在 CPU 上乘以 π/180 转弧度。
eˣ / ln：区间归约 + 泰勒/atanh 级数机器码子程序（e^x=2^k·e^r；ln a=e·ln2+ln m）。log=ln/ln10。
xʸ：整数幂用重复相乘（E_POW，精确）；非整数幂 = eˣ 与 ln 组合 exp(y·ln x)——全部在 CPU 上，不再有任何 Math 兜底。
指令集：MOVI/MOV/LD/ST/ADD/SUB/MUL/DIV/SHL/SHR/CMP/条件跳转/CALL/RET/HALT，编码成 32 位机器字，取指-译码-执行。
